## Design Context

### Users
- 한국인 게이머, 모바일 60%+, 주 사용 맥락은 **"밤에 모바일로 구글에서 쿠폰/티어 검색 → 복사 → 게임 복귀"**.
- 2차 사용: 캐릭터/몬스터링/세계관 탐색. 낮에 PC·태블릿으로 긴 세션.
- 정보 소비형 독자. 숙련 게이머는 정보 밀도에 관대하되 **신뢰감·업데이트 신선도**에 민감.

### Brand Personality
3단어: **환상적 · 몰입 · 밀도**.
톤: 게임 공식(stardive.netmarble.com/ko)의 미드나잇 코즈믹 감성을 공식 IP의 자연스러운 연장처럼 느끼게 하되, **과잉 이펙트(배경 파티클, 자동재생, 사방 애니메이션)는 걷어내고 편집 밀도**로 수렴. 한국적 요소는 개념상 배경에 있으나 표면에 직설적으로 올리지 않음(한자 병기·서예·한지 금지).
감정 목표: 방문자가 "게임 세계관 안에 계속 있다"는 몰입 + "복잡한 정보가 이렇게 정돈되어 있다"는 신뢰.

### Aesthetic Direction
**"별다이브 아카이브(Star-Dive Archive) — 미드나잇 바다 속의 편집 위키"**.
- **다크 온리**. 배경은 순흑이 아닌 미드나잇 네이비 (공식 사이트 `rgb(9,15,31)` 기준).
- **핑크-라벤더 스타더스트**가 악센트 — 공식 사이트의 섹션 헤드라인 그라디언트에서 직접 차용. 단, 우리는 **gradient-text 패턴 금지**(AI slop 탐지기)이므로 solid color로 분해해 사용. 피치 핑크 또는 페리윙클 라벤더 중 하나만 solid 강조.
- 게임 공홈과 같은 톤 가족(family) — 브랜드 연속성 유지 — 다만 정보 밀도는 훨씬 높고 움직임은 훨씬 적다.
- **레이아웃**: 비대칭 편집 그리드. 수평 룰(━━━━━━) 반복. 숫자 자체가 타이포그래피 요소(21·7·0 카운터 스트립).
- **시그니처 모멘트**: 쿠폰 카드 — 80pt 모노 코드, 탭 한 번에 복사+햅틱+미세 상승+페리윙클 ring pulse(200ms). 사이트 전체에서 유일한 과감한 모션.
- 금지: 파티클, 글로우, 그라디언트 텍스트, 글래스모피즘 남용, 자동재생 영상, 배경 애니메이션.

### Typography
검토 순서:
1. 브랜드 3단어: 환상적·몰입·밀도 + 코즈믹 편집.
2. 반사적 선택 (전부 reject): Pretendard, Noto Sans KR, IBM Plex Sans KR, Spoqa Han Sans, Inter, DM Sans, Instrument Serif.
3. 카탈로그 탐색 결과 — **공식 사이트의 폴백 체인에 등장하는 SUIT**는 공식과의 가족성을 자연스럽게 잇는 동시에 Pretendard 대비 사용률이 낮은 한국 gothic. 단일 family + 큰 대비의 웨이트 점프로 편집 밀도를 만든다.

**Primary (본문·UI·디스플레이)**: **SUIT Variable** — 100~900 가변, 한국어·라틴 모두 커버, 공식 사이트 폴백에 내재해 있어 브랜드 패밀리 연속성 확보. 숫자는 `font-variant-numeric: tabular-nums`.
**Signature Mono (쿠폰 코드 전용)**: **Departure Mono** — 레트로 우주 기기 터미널. "star dive" 메타포에 직접 공명. 쿠폰 카드 80pt, 버전 스탬프, 시간 표기에만 사용.

타입 규칙:
- Display(홈 히어로, /coupons H1)는 fluid `clamp(2.25rem, 4.5vw + 1rem, 4.25rem)`, weight 800, tracking `-0.02em`.
- Body/UI는 고정 rem 모듈러 스케일 1.25배 5 스텝: `0.75 / 0.875 / 1 / 1.25 / 1.5625 / 1.953 rem`.
- 본문 line-height 1.65 (다크 배경 보정 +0.05 가산 포함).
- 본문 가로폭 제한 `max-inline-size: 68ch`.
- 웨이트 대비 공격적으로: 800 display ↔ 400 body ↔ 500 meta. 600·700 중간 웨이트 남용 금지.
- 모노는 쿠폰 코드 외에는 사용 금지(의미 희석 방지).

### Color Palette (OKLCH, 공식 사이트 계측 기반 재구성)

계측 샘플:
- body bg `rgb(9,15,31)` ≈ `oklch(0.15 0.043 268)` (미드나잇 네이비)
- accent peach `rgb(248,196,255)` ≈ `oklch(0.86 0.085 335)` (피치 핑크)
- accent periwinkle `rgb(217,213,255)` ≈ `oklch(0.87 0.055 290)` (페리윙클 라벤더)
- deep purple-navy `rgb(28,26,65)` ≈ `oklch(0.24 0.075 285)` (카드 하이라이트)

토큰:
```css
/* 쪽빛 제거 — 코즈믹 미드나잇으로 재정의 */
--bg-base:       oklch(0.14 0.045 268);  /* 공식과 톤 맞춤, 살짝 더 딥 */
--bg-surface:    oklch(0.19 0.045 268);
--bg-raised:     oklch(0.24 0.050 280);
--border:        oklch(0.34 0.030 275);
--border-strong: oklch(0.48 0.040 280);

--fg-primary:    oklch(0.97 0.008 280);   /* 따뜻한 화이트 */
--fg-muted:      oklch(0.78 0.020 280);
--fg-dim:        oklch(0.58 0.020 275);

/* Accent — pink/lavender stardust. 둘 중 하나만 solid로 강조 사용. */
--accent:        oklch(0.86 0.085 335);   /* 피치 핑크 — primary CTA·active badge·시그니처 */
--accent-soft:   oklch(0.78 0.070 335);
--accent-alt:    oklch(0.87 0.055 290);   /* 페리윙클 — 쿠폰 링 펄스·호버 링크 */
--accent-deep:   oklch(0.45 0.120 310);   /* hover/focus outline, 어두운 변종 */

--danger:        oklch(0.68 0.18 25);     /* 만료, 긴급 */
--success:       oklch(0.74 0.12 150);    /* active badge 대체 옵션(쓰임 최소) */
```

60–30–10 비중:
- **60%** — `--bg-base` / `--bg-surface` (미드나잇)
- **30%** — `--fg-primary` / `--fg-muted` / `--border` (텍스트·보더)
- **10%** — `--accent` (피치 핑크) — 쿠폰 active 카드 테두리, primary CTA, 방문 중 링크, 쿠폰 복사 ring pulse에만.

**규율**:
- 순흑 (`#000`) 과 순백 (`#fff`) 절대 금지. 전부 네이비 틴트 적용.
- gradient text (`background-clip: text` + gradient) 금지. 공식 사이트가 쓰더라도 우리는 solid 한 색만 사용해 AI slop 탐지를 피함. 대신 display에서 **피치 핑크 solid** 또는 **화이트 solid** 두 가지로 분리해 리듬을 만듦.
- 네온 청록 on 검정, 보라→파랑 그라디언트 배경, glow border 전부 금지.
- 대비 WCAG AA 이상 강제 (`--fg-muted` on `--bg-surface` 측정 통과).

### Spatial & Layout
- 4pt 스케일 시맨틱 토큰: `--space-2xs(4) / xs(8) / sm(12) / md(16) / lg(24) / xl(32) / 2xl(48) / 3xl(64) / 4xl(96)`.
- Sibling 간격은 `gap`, 헤딩 위 여백만 margin.
- 홈은 **비대칭 편집**: 좌측 대형 헤드라인(80~96px) + 우측 메타 카운터 스트립(숫자 타이포) + 전폭 쿠폰 피드.
- 쿠폰 그리드: `grid-template-columns: repeat(auto-fit, minmax(340px, 1fr))`. 모바일 1열, md 2열, lg 3열.
- 캐릭터/로어 상세: 본문 2열 + 우측 스티키 메타(데스크탑). 모바일은 상단 스티키 세그먼트 탭.
- 모든 컴포넌트 `@container` 반응형. 카드 in 사이드바는 viewport가 아닌 컨테이너 기반.
- 본문 max-inline-size 68ch 강제.
- "not everything in cards": 헤드라인·카운터 스트립·본문은 카드 없이 본문 공간에 그대로. 카드는 쿠폰·캐릭터·링 같은 리스트 아이템에만.

### Motion
- 전역 easing `cubic-bezier(0.22, 1, 0.36, 1)` (ease-out-quint), duration 200–280ms.
- **시그니처 모션**: 쿠폰 복사 성공 시 코드가
  1. `translateY(-4px)` + opacity flash (160ms)
  2. 페리윙클 `outline` ring pulse 2회 (200ms)
  3. 코드 → "복사됨 ✓" 라벨 교체 (1.8s 유지 후 복원)
  4. `navigator.vibrate?.(8)` 호출 (지원 시)
- 그 외 모션 최소. stagger 로드 금지 (편집 밀도 원칙과 충돌).
- `prefers-reduced-motion: reduce` 시 시그니처도 제거 (상태만 즉시 전환).
- `transform` / `opacity`만 애니메이트. layout props 금지.
- 호버는 border/tint 미세 전환 (border 색, bg 3~6% 밝기) 만. 스케일 금지.

### Decorative Language
- 수평 룰 `━━━━━━` (1px `--border`) — 섹션 구분에 반복. 글로우·그라디언트 금지.
- 숫자 타이포그래피: 홈 상단 "21 · 7 · 0" 스트립 (캐릭터 수 · active 쿠폰 · 만료 쿠폰). Departure Mono 72pt tabular-nums.
- **한자 병기 레이블 사용하지 않음** (사용자 선호 반영: "한국 전통 ㅇㅈㄹ").
- 배경 텍스처 최소: 전면에 **0.012 opacity 별점 SVG 노이즈**. 정지 상태, 애니메이션 금지.
- 아이콘: 속성 5종(불/얼음/땅/바람/번개)은 커스텀 선화 글리프 + 라벨 병기. 색 단독 구분 금지.
- 배지: active(`--success`) / expired(`--fg-dim` border) / unknown(`--accent` fill) / limited(`--accent-alt` border) / urgent(`--danger`). 토큰 고정.

### Anti-patterns (절대 금지)
1. 네이버 블로그·티스토리 광고판 — 왕관 이모지, 빨강/노랑 박스, 삐까번쩍 CTA, 광고 사이사이 본문
2. AI slop 다크 대시보드 — 네온 청록 on 검정, 보라→파랑 그라디언트 타이틀, glow border, 동일 유리 카드 반복
3. 기업 SaaS 랜딩 — 가운데 정렬 히어로, 아이콘 3개 feature 그리드, Plus Jakarta Sans, 파란 primary
4. 게임 공홈 과잉 이펙트 — 배경 파티클, 자동재생 PV, 사방 움직이는 애니메이션. 공식 사이트를 참고하되 이 레이어는 걷어낸다 ("적당하면 OK" 원칙).
5. 절대 금지 CSS: `border-left/right: >1px` 스트라이프, `background-clip: text` + gradient, 글래스모피즘 남용, 장식용 스파크라인, 쓸데없는 모달.

### Design Principles
1. **편집 밀도 > 여백 남용.** 정보 많은 팬사이트의 생명은 정돈된 밀도. 카드로 다 감싸지 않는다. 쿠폰·캐릭터·링 같은 리스트 아이템만 카드.
2. **피치 핑크는 희소해서 힘을 가진다.** 10% 규율 엄수. 남용은 네온 청록 남용과 동일하다. 쿠폰 active 상태, primary CTA, 복사 ring pulse, 방문 중 링크에만.
3. **쿠폰 복사 UX가 사이트의 시그니처다.** 다른 모든 모션·장식은 이 순간을 방해하지 않는 수준으로 조절.
4. **브랜드 연속성: 공식 톤 가족을 잇되 레이어를 빼고 밀도를 더한다.** 미드나잇 네이비와 스타더스트 핑크-라벤더는 공식에서, 편집 밀도와 코드 시그니처는 우리에게서.
5. **순색·gradient text 금지.** 검정·흰색 전부 네이비 틴트. 그라디언트 텍스트 배제 — AI 지문.
6. **접근성은 조건이 아니라 기본.** 색 단독 구분 금지, 대비 AA 이상, `prefers-reduced-motion` 존중.
