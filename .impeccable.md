## Design Context

### Users
- 한국인 게이머, 모바일 60%+, 주 사용 맥락은 **"밤에 모바일로 구글에서 쿠폰/티어 검색 → 복사 → 게임 복귀"**.
- 2차 사용: 캐릭터/몬스터링/세계관 탐색. 낮에 PC·태블릿으로 긴 세션.
- 정보 소비형 독자. 숙련 게이머는 정보 밀도에 관대하되 **신뢰감·업데이트 신선도**에 민감.

### Brand Personality
3단어: **게이밍 · 밀도 · 신뢰**.
톤: 한국 게이머가 즐겨 쓰는 게임 위키(paimon.moe, prydwen.gg, 나무위키)의 정보 밀도 + 스타다이브 고유의 미드나잇 다크 팔레트. **과잉 이펙트(배경 파티클, 자동재생, 사방 애니메이션)는 전부 제거**, 편집 밀도와 1px 보더로 구조를 잡는다. 핑크·파스텔 완전 배제.
감정 목표: "이걸 만든 사람이 게임 제대로 하는 사람이다" — 밀도 있는 데이터 허브. 스팸 사이트 느낌 절대 없음.

### Aesthetic Direction
**"한국 게이밍 위키 밀도 — 쿨 다크, 아이스 사이안"**.
- **다크 온리**. 배경은 미드나잇 쿨 (oklch 0.11, hue 260 — 이전 버전보다 더 차갑고 어두움).
- **핑크·파스텔은 UI 크롬에서 완전 추방**. 아이스 사이안(hue 220)이 primary accent — active badge, CTA, 복사 버튼, 링크 호버.
- **핫 오렌지(hue 40)**는 만료·긴급에만. **앰버 골드(hue 85)**는 SSR 배지·S티어 라벨에만.
- **레이아웃**: 데스크탑 `220px sidebar + 1fr content` 위키 2컬럼. 모바일은 상단 수평 탭 스트립.
- **시그니처 모멘트**: 쿠폰 복사 — 아이스 사이안 outline ring pulse (180ms) + translateY(-3px) rise. 거대한 pink 버튼 없음.
- 금지: 파티클, 글로우, 그라디언트 텍스트, 글래스모피즘, 자동재생, 배경 애니메이션.

### Typography
**Primary (본문·UI·디스플레이)**: **SUIT Variable** — 100~900 가변, 공식 사이트 폴백 체인 계보.
**Signature Mono (쿠폰 코드 전용)**: **Departure Mono** — 터미널 레트로. 쿠폰 코드 18–22px, 버전 스탬프에만.

타입 규칙:
- **Base font-size: 14px** (html에 고정). Rem 단위는 14px 기준.
- Display 최대 30px (`1.875rem`). `clamp()` hero 금지.
- 타입 스케일: `11 / 12 / 14 / 16 / 20 / 24 / 30px` (0.6875 / 0.75 / 1 / 1.14 / 1.43 / 1.71 / 2.14rem).
- Body line-height 1.45 (조밀한 위키 표준).
- 본문 가로폭 `max-width: 880px` (콘텐츠 영역), 테이블은 전폭.
- 웨이트: 800 heading ↔ 400/500 body ↔ 700 code. 중간 600·700 남용 금지.
- `font-variant-numeric: tabular-nums` 모든 데이터 테이블.
- Tailwind `text-5xl` 이상 클래스 사용 금지.

--bg-surface:    oklch(0.15 0.018 260);  /* 카드 매트 */
--bg-raised:     oklch(0.19 0.020 260);  /* 호버/활성 */
--border:        oklch(0.28 0.015 260);  /* 1px 보더 — 밀도의 핵심 */
--border-strong: oklch(0.42 0.020 260);  /* 탭, 포커스 */

--fg-primary:    oklch(0.96 0.006 260);
--fg-muted:      oklch(0.70 0.012 260);
--fg-dim:        oklch(0.50 0.012 260);

--accent:        oklch(0.76 0.14 220);   /* 아이스 사이안 — primary CTA/active/링크 */
--accent-hot:    oklch(0.72 0.18 40);    /* 핫 오렌지 — 만료/긴급 */
--accent-gold:   oklch(0.82 0.13 85);    /* 앰버 골드 — SSR/S티어 */

/* 원소 도트 전용 */
--el-fire:    oklch(0.70 0.18 35);
--el-ice:     oklch(0.80 0.12 220);
--el-earth:   oklch(0.72 0.10 100);
--el-wind:    oklch(0.80 0.11 150);
--el-thunder: oklch(0.80 0.15 85);

**규율**:
- 핑크/피치 (#335 계열) UI 크롬 완전 금지. 캐릭터 PNG 내부 픽셀에만 존재 허용.
- 순흑 (`#000`) / 순백 (`#fff`) 금지. 전부 oklch 틴트.
- gradient text 금지.
- 네온 청록 glow, 보라→파랑 그라디언트 배경 금지.
- 대비 WCAG AA 이상 강제.

### Spatial & Layout
- 컴팩트 스케일 토큰: `2px / 4px / 6px / 8px / 12px / 16px / 24px`.
- 기본 컴포넌트 간격: `8px` (gap-3), 섹션 구분: `16px` (gap-5).
- **전역 레이아웃**: 48px sticky top bar + `220px sidebar + 1fr content` (≥1024px). 모바일은 sidebar → 수평 탭 스트립.
- **홈**: 히어로 미니 스트립(140px) + 쿠폰 수평 스크롤 스트립 + 캐릭터/티어 2컬럼 패널. 1440×900에서 스크롤 없이 전체 표시.
- **쿠폰**: 테이블 (카드 그리드 아님). fullWidth 레이아웃.
- **캐릭터 상세**: `2fr portrait + 3fr info` 2컬럼. 모바일 단열.
- 보더가 구조의 primary 언어. 박스 섀도우 금지. `border-radius: 2px` 기본 (rounded-sm).
- `rounded-xl` / `rounded-2xl` / `rounded-full` 금지 (배지 제외, `border-radius: 2px`).
- text-align: center는 히어로·빈 상태에만.

### Motion
- 전역 easing `cubic-bezier(0.22, 1, 0.36, 1)` (ease-out-quint), duration 200–280ms.
- **시그니처 모션**: 쿠폰 복사 성공 시 코드가
  1. `translateY(-4px)` + opacity flash (160ms)
  2. 페리윙클 `outline` ring pulse 2회 (200ms)
  3. 코드 → "복사됨 ✓" 라벨 교체 (1.8s 유지 후 복원)
  4. `navigator.vibrate?.(8)` 호출 (지원 시)
- 그 외 모션 최소. stagger 로드 금지 (편집 밀도 원칙과 충돌).
- `prefers-reduced-motion: reduce` 시 시그니처도 제거 (상태만 즉시 전환).
- `transform` / `opacity`만 애니메이트. layout props 금지.
- 호버는 border/tint 미세 전환 (border 색, bg 3~6% 밝기) 만. 스케일 금지.

### Decorative Language
- 수평 룰 `━━━━━━` (1px `--border`) — 섹션 구분에 반복. 글로우·그라디언트 금지.
- 숫자 타이포그래피: 홈 상단 "21 · 7 · 0" 스트립 (캐릭터 수 · active 쿠폰 · 만료 쿠폰). Departure Mono 72pt tabular-nums.
- **한자 병기 레이블 사용하지 않음** (사용자 선호 반영: "한국 전통 ㅇㅈㄹ").
- 배경 텍스처 최소: 전면에 **0.012 opacity 별점 SVG 노이즈**. 정지 상태, 애니메이션 금지.
- 아이콘: 속성 5종(불/얼음/땅/바람/번개)은 커스텀 선화 글리프 + 라벨 병기. 색 단독 구분 금지.
- 배지: active(`--success`) / expired(`--fg-dim` border) / unknown(`--accent` fill) / limited(`--accent-alt` border) / urgent(`--danger`). 토큰 고정.

### Anti-patterns (절대 금지)
1. 네이버 블로그·티스토리 광고판 — 왕관 이모지, 빨강/노랑 박스, 삐까번쩍 CTA, 광고 사이사이 본문
2. AI slop 다크 대시보드 — 네온 청록 on 검정, 보라→파랑 그라디언트 타이틀, glow border, 동일 유리 카드 반복
3. 기업 SaaS 랜딩 — 가운데 정렬 히어로, 아이콘 3개 feature 그리드, Plus Jakarta Sans, 파란 primary
4. 게임 공홈 과잉 이펙트 — 배경 파티클, 자동재생 PV, 사방 움직이는 애니메이션. 공식 사이트를 참고하되 이 레이어는 걷어낸다 ("적당하면 OK" 원칙).
5. 절대 금지 CSS: `border-left/right: >1px` 스트라이프, `background-clip: text` + gradient, 글래스모피즘 남용, 장식용 스파크라인, 쓸데없는 모달.

### Design Principles
1. **편집 밀도 > 여백 남용.** 정보 많은 팬사이트의 생명은 정돈된 밀도. 카드로 다 감싸지 않는다. 쿠폰·캐릭터·링 같은 리스트 아이템만 카드.
2. **피치 핑크는 희소해서 힘을 가진다.** 10% 규율 엄수. 남용은 네온 청록 남용과 동일하다. 쿠폰 active 상태, primary CTA, 복사 ring pulse, 방문 중 링크에만.
3. **쿠폰 복사 UX가 사이트의 시그니처다.** 다른 모든 모션·장식은 이 순간을 방해하지 않는 수준으로 조절.
4. **브랜드 연속성: 공식 톤 가족을 잇되 레이어를 빼고 밀도를 더한다.** 미드나잇 네이비와 스타더스트 핑크-라벤더는 공식에서, 편집 밀도와 코드 시그니처는 우리에게서.
5. **순색·gradient text 금지.** 검정·흰색 전부 네이비 틴트. 그라디언트 텍스트 배제 — AI 지문.
6. **접근성은 조건이 아니라 기본.** 색 단독 구분 금지, 대비 AA 이상, `prefers-reduced-motion` 존중.
